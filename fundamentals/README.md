# Fundamentals

## Introduction

This section focuses on building a strong foundation in programming by practicing essential concepts and problem-solving techniques. The goal is to develop clear logical thinking, understand core constructs, and write clean and efficient code. These fundamentals form the base for advanced topics such as data structures, algorithms, and software development.

---

## Topics Covered

* Variables and Data Types
* Conditional Statements (if-else, switch)
* Loops (for, while)
* Functions and Modular Programming
* Arrays and Lists
* Strings and String Manipulation
* Basic Input/Output Handling
* Mathematical Computations
* Logical Problem Solving
* Pattern-based Problems

---

## Problem Statements

* Check whether a number is prime
* Find the factorial of a number
* Reverse a given number
* Check if a number is a palindrome
* Generate Fibonacci sequence up to N terms
* Find the largest and smallest element in an array
* Calculate the sum and average of array elements
* Count the number of vowels and consonants in a string
* Check if a string is a palindrome
* Sort an array without using built-in functions
* Find the second largest element in an array
* Merge two arrays and remove duplicates
* Count frequency of elements in an array
* Print different star patterns (triangle, pyramid, etc.)
* Find GCD and LCM of two numbers
* Check whether a year is a leap year
* Convert temperature between Celsius and Fahrenheit
* Find the sum of digits of a number
* Check if a number is an Armstrong number
* Count words in a sentence
* Remove spaces from a string
* Find the occurrence of a substring in a string


This section is intended for continuous practice. Problems will be added and updated over time to strengthen core programming skills and improve problem-solving ability.
